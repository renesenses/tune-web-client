# Screenshots pour le Guide Visuel Tune

Captures d'ecran a fournir pour le guide utilisateur sur mozaiklabs.fr.
Resoluton recommandee : **1440x900** (ou 1280x720 minimum).
Format : **PNG** de preference.

## 1. Accueil
- **Fichier** : `Tune_Accueil.png`
- **Contenu** : Page d'accueil avec le message de bienvenue, les statistiques (albums, artistes, pistes), la barre de recherche, et les sections "Recemment joue" / "Recemment ajoute"

## 2. Bibliotheque — Grille Albums
- **Fichier** : `Tune_Bibliotheque.png`
- **Contenu** : Vue Bibliotheque avec l'onglet Albums actif, la grille d'albums visible, et les filtres de qualite (DSD, Hi-Res, CD) visibles en haut
- **Note** : Montrer aussi les onglets Albums/Artistes/Pistes/Genres

## 3. Bibliotheque — Detail Album
- **Fichier** : `Tune_Album_Detail.png`
- **Contenu** : Vue detail d'un album avec pochette, infos (titre, artiste, annee, genre), bouton "Lire l'album", bouton "Editer", et liste des pistes avec badges de format

## 4. Lecture en cours (Now Playing)
- **Fichier** : `Tune_Lecture.png`
- **Contenu** : Vue "Lecture en cours" avec un morceau en lecture : grande pochette, titre, artiste, album, badge de qualite audio, barre de progression, controles shuffle/repeat, file d'attente "A suivre"

## 5. File d'attente (Queue)
- **Fichier** : `Tune_Queue.png`
- **Contenu** : Vue "File d'attente" avec plusieurs morceaux dans la queue, montrant les handles de drag-drop, le morceau en cours surligne, les boutons de suppression

## 6. Radios Live
- **Fichier** : `Tune_Radios.png`
- **Contenu** : Vue "Radios Live" avec les stations FIP/Radio France (avec leurs logos), les filtres par genre, les boutons favori/lecture/editer/supprimer visibles
- **Note** : Montrer aussi les boutons "Ajouter une radio" et "Importer M3U/PLS"

## 7. Streaming — Qobuz ou TIDAL
- **Fichier** : `Tune_Streaming.png`
- **Contenu** : Vue d'un service de streaming (Qobuz ou TIDAL) avec la barre de recherche, les carrousels d'albums recommandes, les sections editoriales

## 8. Favoris
- **Fichier** : `Tune_Favoris.png`
- **Contenu** : Vue "Favoris" avec les onglets Pistes/Albums/Artistes, montrant quelques favoris

## 9. Recherche
- **Fichier** : `Tune_Recherche.png`
- **Contenu** : Vue "Recherche" avec un terme de recherche saisi et des resultats affiches (albums + pistes + artistes, de sources locale et streaming)

## 10. Parametres
- **Fichier** : `Tune_Parametres.png`
- **Contenu** : Vue "Parametres" avec la sante du serveur (tous les indicateurs OK), les stats de la bibliotheque, les boutons Scanner/Rescanner, les emplacements musicaux, la section Appareils
- **Note** : Scroller pour montrer aussi la section Services de streaming si possible

## 11. Metadonnees
- **Fichier** : `Tune_Metadonnees.png` (DEJA FOURNI)
- **Contenu** : Vue "Metadonnees" avec les indicateurs de completude, les filtres, les boutons d'action
- **Statut** : OK

## 12. Zones
- **Fichier** : `Tune_Zones.png`
- **Contenu** : Vue "Zones" montrant les zones configurees avec leurs types (DLNA, AirPlay, Local), volumes, et options de groupement

## 13. Playlists
- **Fichier** : `Tune_Playlists.png`
- **Contenu** : Vue "Playlists" montrant la liste des playlists avec le nombre de pistes, et si possible une playlist ouverte avec sa liste de pistes

---

## Instructions

1. Ouvrir le web client Tune dans un navigateur (Chrome de preference)
2. Redimensionner la fenetre a **1440x900** pixels
3. Pour chaque vue, naviguer vers l'ecran correspondant
4. Faire une capture d'ecran (Cmd+Shift+4 puis espace sur macOS, ou outil de capture)
5. Sauvegarder en PNG dans `~/Downloads/`
6. Me les envoyer pour mise a jour du guide visuel
